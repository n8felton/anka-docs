Anka VM Templates support the following macOS versions:

| Version Number | Version Name |
| --- | --- |
| **13.x** | macOS Ventura |
| **12.x** | macOS Monterey |

{{< hint warning >}}
Creating 13.x VMs requires that Xcode >= 14.0.1 is installed. This is a requirement from Apple at the moment.
{{< /hint >}}
