Anka VM Templates support the following macOS versions:

| Version Number | Version Name |
| --- | --- |
| **13.x** | macOS Ventura |
| **12.x** | macOS Monterey |

{{< hint warning >}}
If using a Monterey host, creating a 13.x VM requires 14.x Xcode is installed to avoid an Apple bug.
{{< /hint >}}
